# 💱 CurrencyX – Real-Time Currency Converter

CurrencyX is a modern desktop application built with Python and Tkinter that provides real-time currency conversion with a clean dark-themed interface.  
It includes live exchange rates, multi-currency conversion, favorites management, conversion history, and graphical rate trends.

---

## ✨ Features

- 🔄 Real-time currency conversion
- 🌍 Supports multiple international currencies
- 📊 Live exchange rates viewer
- 🔀 Multi-currency conversion
- ⭐ Favorite currency pairs
- 🕘 Conversion history tracking
- 📈 Rate trend charts using Matplotlib
- 💾 Offline cache support
- 📤 Export history to CSV and JSON
- 🎨 Modern dark-themed UI

---

## 🛠️ Technologies Used

- Python
- Tkinter
- Matplotlib
- JSON & CSV
- ExchangeRate-API

---

Install required dependencies:

```bash
pip install matplotlib
```

---

## ▶️ Run the Application

```bash
python main.py
```
